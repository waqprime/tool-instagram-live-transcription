❓ よくある質問

気になる疑問にお答えします

💰 アプリは無料ですか？

はい、完全無料です。個人利用・商用利用ともに自由にご利用いただけます。

🌐 インターネット接続は必要ですか？

動画のダウンロード時と初回起動時（Whisperモデルのダウンロード）に必要です。文字起こし処理自体はオフラインで実行されます。

⏱️ どのくらい時間がかかりますか？

5分の動画なら約2〜3分、30分の動画なら約10〜15分で完了します。選択したWhisperモデルによって変動します。

🎯 精度を上げるには？

より大きなWhisperモデル（Small, Medium, Large）を選択するか、音声がクリアな動画を選ぶことで精度が向上します。

🔒 プライバシーは大丈夫？

すべての処理はあなたのPC上で実行されます。動画や音声ファイルは外部サーバーに送信されません。完全にプライベートです。
